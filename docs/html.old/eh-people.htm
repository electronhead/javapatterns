First Person

Gary Beaver (beaver@electronhead.com) accepts full responsibility for all ElectronHead pages.

Nutshell:

  • Self-employed software consultant since 1992, off and on since 1984.
  • Started programming in 1976. Two year hiatus selling tortillas 1988-90.
  • Lots of high-level language programming (APL, Scheme, Smalltalk, Dylan, Java, Access Basic, WyScript).
  • Lots of horizontal software design and development, primarily in database arena.
  • Lots of vertical software design and development, primarily in human resource and asset management areas.
  • Current projects include actuarial programming and a speculative effort written in Java.
Examples of horizontal software projects:
  • Designed and implemented an RDBMS and its associated (algebraic) query language (for a computer leasing company).
  • Designed and helped implement a relational database api for a proprietary scripting language, WyScript (trademark Watson Wyatt WorldWide).
  • Designed and implemented RDBMS-based persistent object mechanisms for both Smalltalk and Java.
Examples of vertical software projects:
  • Helped to design and implement a pension administration system authoring environment, with responsibilities ranging from system-level to domain-specific design and implementation.
  • Implemented and helped to design a prototype asset management system that managed the life cycles of leased and purchased software and hardware assets.
  • Designed and implemented additional features for existing pension administration systems.
  • Implemented actuarial calculators for defined benefit pension plans.
Current Java project:
  • Building software to support the rapid design and delivery of ultra-thin inter/intra/extra-net applications.
